```
unction fib(n) {
  var a = 1, b = 1;
  var tmp;
  while (--n >= 0) {
    tmp = a;
    a += b;
    b = tmp;
  }
  return a;
}
 
document.write(fib(10));
```
**上面这段代码和文章开始到现在的所有的代码块都是用代码块语法实现的**
**其代码如下：**
